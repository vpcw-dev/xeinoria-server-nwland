# xeinoria-server-nwland

Depot public des scripts Skript specifiques au serveur **nwland** du reseau Xeinoria.

## Ce repo contient

- les scripts propres au serveur nwland
- la reference vers les scripts partages via le dossier `global/` (submodule)

## Ce repo ne contient pas

- scripts prives/staff sensibles
- secrets, mots de passe, tokens

## Dependances

La liste complete des addons et la doc de contribution globale sont maintenues dans:
- https://github.com/Xeinoria-Studio/xeinoria-scripts-global
